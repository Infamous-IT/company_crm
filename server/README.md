# What we used for developing:

1. JWT & Passport & BCrypt
2. TypeORM (PostgreSQL)
3. Nest.js
4. Class-Transformer & Class-Validator
5. Mapped Typed

# For starting you need to:

Clone the repository
```bash
$ git clone https://github.com
```

Use the following command
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
